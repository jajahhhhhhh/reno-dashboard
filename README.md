## 🚀 วิธีขึ้น GitHub Pages (10 นาที)

### ขั้นตอนที่ 1 — สมัคร GitHub
1. ไปที่ **github.com**
2. กด **Sign up** → ใส่ email + password + username
3. ยืนยัน email

### ขั้นตอนที่ 2 — สร้าง Repository
1. กด **+** มุมขวาบน → **New repository**
2. ตั้งชื่อ: `reno-dashboard`
3. เลือก **Public**
4. กด **Create repository**

### ขั้นตอนที่ 3 — อัปโหลดไฟล์
1. ในหน้า repository กด **uploading an existing file**
2. ลาก 3 ไฟล์นี้เข้าไป:
   - `index.html`
   - `dashboard-final.html`
   - `inventory-system.html`
3. กด **Commit changes**

### ขั้นตอนที่ 4 — เปิด GitHub Pages
1. กด **Settings** (แถบบนสุด)
2. เลือก **Pages** (เมนูซ้าย)
3. ใต้ **Source** เลือก **Deploy from a branch**
4. Branch: **main** → Folder: **/ (root)**
5. กด **Save**
6. รอ 2-3 นาที → ได้ URL:

```
https://[username].github.io/reno-dashboard/
```

## 📱 วิธีเปิดบนมือถือ

1. เปิด URL ด้านบนในมือถือ
2. กด **Share → Add to Home Screen**
3. ได้ไอคอนบนหน้าจอ เหมือน App เลย!

---

## 🔄 วิธีอัปเดตข้อมูล

### อัปเดต Dashboard (Kanban / การเงิน)
1. แก้ไขไฟล์ `dashboard-final.html` บนคอม
2. ไปที่ GitHub → คลิกที่ไฟล์ → กด ✏️ (Edit)
3. วางเนื้อหาใหม่ → Commit changes
4. รอ 1-2 นาที → อัปเดตทันที

### อัปเดตผ่าน AI อ่าน LINE (ง่ายที่สุด)
1. เปิด `dashboard-final.html`
2. กด **📁 LINE .txt** หรือ **📋 วางข้อความ**
3. AI อ่านและอัปเดตให้อัตโนมัติ
4. ข้อมูลเซฟใน browser (localStorage) ทันที
